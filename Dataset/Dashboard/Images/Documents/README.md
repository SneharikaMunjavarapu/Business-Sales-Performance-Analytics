# Business Sales Performance Analytics

## Objective

Analyze business sales performance using Tableau to identify trends, top-performing categories, regions, and products.

## Tools Used

* Tableau Public
* Microsoft Excel

## KPIs

* Total Sales: 2,297,201
* Total Profit: 286,397
* Total Orders: 5,009
* Total Customers: 793

## Dashboard Features

* Sales Trend Analysis
* Category Sales Analysis
* Category Profit Analysis
* Region Sales Analysis
* Region Profit Analysis
* Top 10 Products

## Key Insights

* Sales remained relatively stable throughout the observed period.
* Technology generated the highest sales among all categories.
* West region recorded the highest sales.
* Canon imageCLASS 2200 Advanced Copier was one of the top-performing products.

## Recommendations

* Focus on high-performing categories.
* Improve performance in low-sales regions.
* Promote top-selling products.
